📊 Projeto Power BI – Análise Financeira e de Vendas
📌 Visão Geral
Este projeto tem como objetivo realizar uma análise financeira e de vendas utilizando o Microsoft Power BI, explorando indicadores importantes como soma de vendas, preço médio de venda, desempenho por produto, segmento de mercado e análise temporal (ano e mês).
O dashboard permite uma visão clara e interativa dos dados, auxiliando na tomada de decisão estratégica e na identificação de padrões e oportunidades de negócio.

🎯 Objetivos do Projeto

Analisar o desempenho de vendas por produto
Avaliar o impacto das vendas por segmento de mercado
Comparar métricas ao longo do tempo (ano e mês)
Identificar produtos e segmentos mais rentáveis
Criar um dashboard intuitivo, interativo e visualmente claro


🛠️ Ferramentas Utilizadas

Microsoft Power BI
Power Query (tratamento e limpeza de dados)
DAX (Data Analysis Expressions) para criação de métricas
Modelo de Dados Relacional


📈 Principais Métricas e Análises


Soma das Vendas por Produto


Média de Preço de Venda por Produto


Soma de Vendas por Ano, Mês e Segmento


Comparação entre produtos como:
